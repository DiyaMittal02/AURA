# AURA - Backend Orchestrator

The central hub that manages style logic, persona fusion, and data routing.

## Responsibilities
- **Quiz Processing**: Calculates archetype scores from user input.
- **Persona Fusion**: Combines AI results with quiz data using a 16x4 matrix.
- **AI Integration**: Proxy communication with the Python AI service.
- **Data Serving**: provides quiz questions and mock product catalog.

## Tech Stack
- Node.js
- Express.js
- Axios

## Setup
```bash
npm install
node server.js
```
