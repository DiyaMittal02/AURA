# AURA - Frontend

This is the React-based frontend for the AURA project.

## Key Components
- **PaletteScanner**: Interface for uploading photos and interacting with the AI service.
- **Quiz**: The Style DNA assessment.
- **Feed**: Personalized product discovery.
- **Aurora/LiquidEther**: Custom-built CSS animations for a premium feel.

## Tech Stack
- React + Vite
- Framer Motion (Animations)
- Axios (API Communication)

## Setup
```bash
npm install
npm run dev
```
