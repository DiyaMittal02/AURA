# AURA - AI Service

This service handles the seasonal color analysis using Deep Learning.

## Features
- **ResNet18 Model**: Classifies images into one of 4 seasonal palettes (Spring, Summer, Autumn, Winter).
- **Flask API**: Simple endpoint for image processing and prediction.

## Tech Stack
- Python
- PyTorch & Torchvision
- Flask

## Setup
```bash
pip install -r requirements.txt
python api.py
```
